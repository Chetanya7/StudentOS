# assignments

This feature owns the assignments tab and everything needed to track coursework.

## Responsibilities

- Render the assignments tab UI.
- Show assignment lists, due dates, progress, and task status.
- Keep assignment state and presentation logic isolated from other features.

## Boundaries

- Assignment-specific widgets and models can live here.
- Notification ingestion should not be implemented here.
- Calendar-specific workflows should not be implemented here.
